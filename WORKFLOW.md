# 内容生产工作流 SOP

一套 5 阶段循环：**选题 → 素材 → 生产 → 分发 → 复盘**。所有产出物落到 `01_~07_` 对应目录。

---

## 阶段 1 · 选题（每周一固定 1h）

**输入**：素材库里的资讯/播客/benchmark + 上周复盘留下的"二次创作机会"。

**SOP**：
1. 打开 `02_素材库/热点追踪/本周.md`，列出本周值得追的 5 个事件 / 趋势。
2. 对每个事件，按"账号 × 角度"矩阵分发：
   - 公众号 → 海外深度长文 / benchmark 横评
   - 小红书 IP 号 → 真人出镜短视频 + 一句话观点
   - 小红书 知识号 → 把公众号长文转 4-6 图文摘要
   - 小红书 评测号 → 产品角度切入 + 软文契机
3. 每条选题填写 [`templates/选题卡.md`](templates/选题卡.md)，存入 `01_选题库/<账号>/YYYYMMDD_标题.md`。
4. 周末复核：未启动的选题超过 14 天直接归档到 `01_选题库/_归档/`。

**产出**：本周排期表，写入 `06_发布日历/YYYY-MM-DD_周.md`。

---

## 阶段 2 · 素材（持续）

**信息源（每日 30min 扫读）**：

| 类别 | 渠道 | 落地位置 |
|------|------|----------|
| 海外科技公司动态 | Anthropic/OpenAI/Google 官 blog、X 上 Sam/Dario/Demis、TechCrunch、The Information | `02_素材库/海外资讯/` |
| 海外长播客 | Lex Fridman、Dwarkesh、Latent Space、No Priors、a16z | `02_素材库/播客笔记/` |
| benchmark / 评测 | LMSYS Arena、SWE-bench、Artificial Analysis、Aider leaderboard | `02_素材库/benchmark数据/` |
| 独立开发者 | indie hackers、ProductHunt、X 头部 PM | `02_素材库/海外资讯/indie/` |
| 国内热点 | 即刻、X 中文圈、虎嗅、36 氪 | `02_素材库/热点追踪/` |

**素材卡模板**：[`templates/素材卡.md`](templates/素材卡.md)

> 重要：**任何素材都要带原始链接 + 时间戳**，避免后续追溯困难和事实错误。

## 阶段 3 · 生产

### 3.1 公众号长文（每周 2-3 篇）

1. 用 `wechat-article-writer` skill 起初稿（提供选题卡 + 关键素材）。
2. 落到 `03_草稿/公众号/`。
3. 自检清单（写在草稿底部）：
   - [ ] 开头 100 字内出现"钩子"（数据/反常识/冲突）
   - [ ] 中段至少 1 个 benchmark 对比表 或 时间线表
   - [ ] 至少 1 处 call-to-action 引导私域（公众号菜单 / 进群 / 加微）
   - [ ] 关键事实带原始链接
   - [ ] 标题 3 个版本，选爆款向最强的
4. 用 `wechat-studio` 排版 + 上传图片 + 存草稿。

### 3.2 小红书 个人 IP 号（视频为主，每周 3-4 条）

1. 选题卡 → 写 30s/60s 口播脚本，存 `03_草稿/小红书_个人IP/`。
2. 模板：[`templates/小红书_视频脚本.md`](templates/小红书_视频脚本.md)
3. 拍摄要点写在脚本底部（机位 / 道具 / 镜头序列）。
4. 封面：`atutun-xhs-cover` skill 生成提示词 → 出图 → 存 `05_封面/`。

### 3.3 小红书 知识/资讯号（图文为主，每周 5-7 条）

1. 来源 1：公众号长文 → 拆 1 主图 + 4-6 内页图文摘要。
2. 来源 2：长播客 → 5 个金句 / 1 个核心观点图。
3. 模板：[`templates/小红书_图文.md`](templates/小红书_图文.md)
4. 风格保持纯客观，**不带个人结论**，引流走主页/简介挂的公众号。

### 3.4 小红书 产品评测号（图文+视频混合，每周 3-5 条）

1. 选题分两类：benchmark 解读 / 产品上手测评。
2. 模板：[`templates/小红书_产品评测.md`](templates/小红书_产品评测.md)
3. 中转站软文必须标注合作类型（在内部记录里，不发出去）。

---

## 阶段 4 · 分发

**发布日历**：每周日晚把下周排期写入 `06_发布日历/YYYY-MM-DD_周.md`。模板：[`templates/周排期.md`](templates/周排期.md)

**联动策略**：
- 公众号长文发出后 → 当天小红书知识号同步精简版图文 → 个人 IP 号当晚拍 30s 口播预热私域。
- 评测号 benchmark 文 → 公众号同主题深度版第二天补发。
- 私域钩子（资源包 / 提示词集 / 表格）放 `08_私域转化/钩子/`，文末 / 评论区固定话术。

---

## 阶段 5 · 复盘（每周日 30min）

1. 数据填到 `07_数据复盘/YYYY-MM-DD_周报.md`。模板：[`templates/周报.md`](templates/周报.md)
2. 必看指标：
   - 公众号：阅读、在看、私域转化（加微/进群）
   - 小红书：曝光、点击率、互动率、涨粉、私信
3. 每周必产出 3 件事：
   - **爆款拆解**：本周最好的一条，拆为何爆（选题 / 钩子 / 标题 / 封面 / 时机）
   - **失败复盘**：本周最差的一条，归因
   - **二次创作清单**：哪条值得改造投到其他账号

---

## 跨账号联动模板

| 公众号 1 篇 | → 知识号 1 帖 | → 个人 IP 号 1 视频 | → 评测号 1 帖 |
|-------------|---------------|----------------------|---------------|
| 海外公司动态深度 | 4 张图摘要 | 我的观点 30s | 受影响产品的评测角度 |
| 播客拆解 | 5 金句卡片 | 我对核心观点的反应 | 涉及产品的实测 |
| benchmark 横评 | 榜单图 + 结论 | 我用第一名的实操 | 详细评测 + 软文位 |
