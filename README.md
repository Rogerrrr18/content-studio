# content-studio

个人媒体矩阵运营工作台。覆盖公众号 + 小红书三个矩阵号的选题、生产、分发、复盘、私域转化全流程。

## 矩阵账号定位

| 平台 | 账号 | 定位 | 主要内容 | 转化目标 |
|------|------|------|----------|----------|
| 微信公众号 | 主号 | 海外科技深度 + 热点 + benchmark | 海外公司动态/播客拆解/产品测评长文 | 私域引流（社群/1对1） |
| 小红书 | 个人 IP 号 | 真人出镜 + 教程 + 观点 | AI 工具使用教程视频 + 行业观点 | 商单 / 合作 / 个人影响力 |
| 小红书 | 知识/资讯号 | 纯客观知识中转 | 海外播客摘要 + 公众号长文图文化 | 涨粉 / 矩阵引流 |
| 小红书 | 产品评测号 | 产品经理 + benchmark 视角 | benchmark 垂类 / 中转站软文 | 软文变现 / 选品合作 |

详细 SOP 见 [`WORKFLOW.md`](WORKFLOW.md)。

## 目录结构

```
01_选题库/        按账号分类的选题池（草创 → 待写 → 已用）
02_素材库/        海外资讯、播客笔记、benchmark 数据、热点追踪
03_草稿/          按账号分类的稿件草稿
04_成稿/          定稿后的最终版本，按平台归档
05_封面/          封面成品 + 封面提示词
06_发布日历/      周/月排期 + 跨账号联动表
07_数据复盘/      每篇阅读/点赞/涨粉/转化数据
08_私域转化/      引流路径 / 钩子素材 / 私域 SOP
templates/        各账号 + 各内容类型的模板
scripts/          自动化脚本（抓资讯/转格式/批量改图）
```

## 常用 Skill

- **`wechat-article-writer`** — 公众号文章自动化写作（搜资料 / 撰稿 / 爆款标题 / 排版）
- **`wechat-studio`** — Markdown 转公众号 HTML、上传图片、发草稿、小绿书
- **`atutun-xhs-cover`** — 阿囤囤风格小红书封面提示词（已通过本工作流安装）
- **`competitor-research`** — 对标账号 / 同行内容分析
- **`customer-research`** — 私域用户研究 / 评论挖掘
